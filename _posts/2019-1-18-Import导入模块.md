# 四种导入模块的方法

1. **from 模块名 import 方法名** 
2. **import 模块名**
3. **from 模块名 import \*** ：导入模块中所有的方法
4. **import 模块1， 模块2**
5. **from 模块名 import 方法名1， 方法名2**
6. **import 模块名 as 别名** ：给导入的模块起一个别名
