# AWT

#### 1、介绍

- 包含了很多的**类和接口**，用于实现 GUI
- 元素：**窗口，按钮，文本框**
- 存在在**java.awt**包中

![AWT.png](https://raw.githubusercontent.com/JTXYH/blog_comment/master/blog_imgs/AWT.png)

#### 2、使用 Frame

```java
// 创建一个Frame对象
java.awt.Frame frame = new java.awt.Frame("第一个Frame程序");

// 设置可见性
frame.setVisible(true);

/*
    设置显示位置
    frame.setLocation(200, 200);
    设置窗口大小
    frame.setSize(400, 400);
*/

// 同时设置窗口弹出的显示位置和大小
frame.setBounds(200, 200, 400, 500);


// 设置背景颜色
frame.setBackground(Color.yellow);
```

#### 2、使用 Panel

```java
Frame frame = new Frame("使用Panel");
Panel panel = new Panel();
// 设置布局
frame.setLayout(null);

// 设置frame显示的坐标和大小
frame.setBounds(200,200,500,500);
frame.setBackground(new Color(255, 224, 0));

// 设置Panel坐标，对应的是frame的布局
panel.setBounds(50,50,300,300);
panel.setBackground(new Color(0, 121, 163));

// 添加Panel到Frame中
frame.add(panel);

// 设置Frame的可见性
frame.setVisible(true);

// 自定义一个窗口监听事件，点击×可以关闭
frame.addWindowListener(new WindowAdapter() {
    // 窗口关闭的时候需要做的事情
    @Override
    public void windowClosing(WindowEvent e) {
        // 关闭程序
        System.exit(0);
    }
});
```

#### 3、布局

1. **流式布局**

```java
Frame frame = new Frame();
Button button1 = new Button("button1");
Button button2 = new Button("button2");
Button button3 = new Button("button3");

frame.setVisible(true);
frame.setBounds(200,200,400,400);
// 设置为流式布局，默认居中显示
// frame.setLayout(new FlowLayout());
// 设置左边显示
// frame.setLayout(new FlowLayout(FlowLayout.LEFT));
// 设置右边显示
frame.setLayout(new FlowLayout(FlowLayout.RIGHT));

// 添加按钮到Frame中去
frame.add(button1);
frame.add(button2);
frame.add(button3);

```

2. **东西南北中布局**

```java
Frame frame = new Frame("东西南北中布局");
Button east = new Button("East");
Button west = new Button("West");
Button south = new Button("South");
Button north = new Button("North");
Button center = new Button("Center");

// 设置布局
frame.add(east,BorderLayout.EAST);
frame.add(west,BorderLayout.WEST);
frame.add(south,BorderLayout.SOUTH);
frame.add(north,BorderLayout.NORTH);
frame.add(center,BorderLayout.CENTER);

frame.setSize(500,500);
frame.setVisible(true);
```

3. **表格布局**

```java
Frame frame = new Frame("表格布局");
Button button1 = new Button("button1");
Button button2 = new Button("button2");
Button button3 = new Button("button3");
Button button4 = new Button("button4");
Button button5 = new Button("button5");

// 设置3行2列
frame.setLayout(new GridLayout(3,2));

frame.add(button1);
frame.add(button2);
frame.add(button3);
frame.add(button4);
frame.add(button5);

// java函数，自动设置布局，不用设置大小，显示在左上角
frame.pack();
frame.setVisible(true);
```
